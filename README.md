#### Previewing the App
Open up `index.html` in your browser.

![A preview of the app](screenshot.png)

#### Development Requirements
- Node.js (v5.4.1)
- Bower (v1.7.7)

#### Install Dependencies
```shell
npm install
bower install
```

#### Compile Sass
```shell
grunt sass

# or, if watching for changes to src/*.scss

grunt
```
